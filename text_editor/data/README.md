# Data Files

This directory contains static data assets used by the application logic.

## Files

- **`languages.json`**: A JSON database defining keywords, functions, types, and other syntax elements for supported programming languages.
  - This file is used by the `AutoCompleter` and potential future syntax analysis tools to provide intelligent suggestions and highlighting.
  - Supported languages key include: `python`, `html`, `css`, `javascript`, `java`, `cpp`, `sql`, `typescript`, `go`, `rust`, `php`, `ruby`.
