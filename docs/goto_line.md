# Satıra Git (Go To Line)

Büyük dosyalarda belirli bir satıra hızlıca ulaşmanızı sağlayan navigasyon aracı.

## Özellikler

*   **Hızlı Erişim:** Satır numarası girerek anında o satıra odaklanma.
*   **Geçerlilik Kontrolü:** Dosya sınırları dışındaki satır numaraları için uyarı.

## Kullanım

`Ctrl + G` kısayolu veya Düzen menüsü üzerinden erişilebilir.
