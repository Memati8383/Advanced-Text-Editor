# HTML Snippet'ları (HTML Snippets)

Bu dokümantasyon, hızlı taslak oluşturma için editördeki tüm HTML snippet'larını listeler.

## Temel Snippet'lar
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `html5` | HTML5 İskeleti | Tam belge şablonu |
| `div` | Sınıf (class) içeren Div | `<div class="class">...</div>` |
| `btn` | Buton | `<button type="button">...</button>` |
| `script` | Script etiketi | `<script src="main.js"></script>` |
| `link` | Link etiketi (CSS) | `<link rel="stylesheet" href="style.css">` |
| `meta_seo` | SEO Meta etiketleri | Açıklama, Anahtar Kelimeler ve Yazar etiketleri |
