# CSS Snippet'ları (CSS Snippets)

Bu dokümantasyon, yaygın mizanpajlar ve stillendirme için editördeki tüm CSS snippet'larını listeler.

## Temel Snippet'lar
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `flex` | Flexbox ortalama | `display: flex; ...` |
| `grid` | Grid mizanpajı | `display: grid; ...` |
| `glass" | Cam efekti (Glassmorphism) | `backdrop-filter: blur(...);` |
| `media` | Medya sorgusu (Media query) | `@media (max-width: 768px) { ... }` |
| `reset` | CSS Sıfırlama (Reset) | Evrensel `*` seçici sıfırlaması |
