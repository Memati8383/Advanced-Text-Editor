# Typescript Snippet'ları (Typescript Snippets)

Bu dokümantasyon, arabirim (interface) ve tür (type) tanımlamaları için editördeki tüm Typescript snippet'larını listeler.

## Temel Snippet'lar
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `interface` | Arabirim (Interface) tanımı | `interface Name { prop: string; }` |
| `type` | Tür (Type) tanımı | `type Name = { prop: string; };` |
| `enum` | Enum tanımı | `enum Name { Value = 'Value' }` |
