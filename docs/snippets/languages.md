# Programlama Dillerine Özgü Snippet'lar (Java, C++, Rust, Go, SQL)

Bu dokümantasyon, çeşitli programlama dilleri için küçük snippet koleksiyonlarını gruplandırır.

## Java
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `main` | Java Ana (Main) sınıfı | `public class Main { ... }` |
| `sout` | Konsola yazdırma | `System.out.println(...);` |
| `for` | Standart for döngüsü | `for (int i=0; ...) { ... }` |

## C++ (CPP)
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `main` | C++ Ana (Main) fonksiyonu | `#include <iostream> ...` |
| `cout` | Standart çıktı | `std::cout << ... << std::endl;` |

## Rust
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `fn` | Rust fonksiyonu | `fn name(params) -> ReturnType { ... }` |
| `main` | Rust ana fonksiyonu | `fn main() { ... }` |
| `println` | println makrosu | `println!("{:?}", val);` |

## Go
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `main` | Go ana fonksiyonu | `package main ... func main() { ... }` |
| `func` | Go fonksiyonu | `func name(params) error { ... }` |

## SQL
| Önek (Prefix) | Açıklama | Örnek İçerik |
| :--- | :--- | :--- |
| `select` | SELECT ifadesi | `SELECT * FROM table WHERE ...` |
| `create` | CREATE TABLE ifadesi | `CREATE TABLE table_name ( ... );` |
