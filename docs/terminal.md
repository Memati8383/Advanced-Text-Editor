# Terminal

Uygulama içerisinden komut satırı işlemlerini yürütmenizi sağlayan gömülü terminal.

## Özellikler

*   **Entegre Çalışma:** Editörden çıkmadan komut çalıştırma imkanı.
*   **Kabuk Desteği:** Sistem varsayılan kabuğu (PowerShell, CMD, Bash vb.) ile uyumluluk.
*   **Çoklu Sekme:** Birden fazla terminal oturumu açma desteği (destekleniyorsa).
*   **Özelleştirilebilir:** Renk ve yazı tipi ayarları.

## Kullanım

Alt panelde yer alır. Menü veya kısayol tuşları ile açılıp kapatılabilir.
