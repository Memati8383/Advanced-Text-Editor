# Markdown Önizleme (Markdown Preview)

Markdown formatındaki dosyaların HTML çıktısını canlı olarak görmenizi sağlar.

## Özellikler

*   **Canlı Önizleme:** Yazdığınız anda güncellenen önizleme penceresi.
*   **GitHub Flavored Markdown:** Yaygın Markdown standartlarını destekler.
*   **Resim ve Link Desteği:** Dış bağlantıları ve görselleri görüntüler.

## Kullanım

Markdown (.md) dosyası açıkken menüden veya sağ tıklama menüsünden önizlemeyi açabilirsiniz.
