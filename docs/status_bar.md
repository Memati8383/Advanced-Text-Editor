# Durum Çubuğu (Status Bar)

Editörün alt kısmında yer alan ve anlık bilgileri gösteren panel.

## Özellikler

*   **İmleç Konumu:** Satır ve sütun numarasını gösterir.
*   **Dosya Dili:** Açık olan dosyanın algılanan programlama dilini belirtir.
*   **Kodlama:** Dosya karakter kodlamasını (UTF-8 vb.) gösterir.
*   **Git Durumu:** (Destekleniyorsa) Git dalı ve değişiklik durumunu gösterir.

## Kullanım

Pencerenin en alt kısmında sürekli olarak görüntülenir.
