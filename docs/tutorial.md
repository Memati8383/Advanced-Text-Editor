# Öğretici Mod (Tutorial Mode)

Yeni kullanıcıların editörü tanımasına yardımcı olan interaktif rehber.

## Özellikler

*   **Adım Adım Rehber:** Temel özellikleri sırasıyla tanıtır.
*   **İnteraktif Görevler:** Kullanıcının özellikleri deneyimlemesini sağlar.
*   **Görsel İşaretçiler:** İlgili menü veya butonu vurgular.

## Kullanım

Yardım menüsünden "Öğretici" seçeneği ile başlatılabilir.
