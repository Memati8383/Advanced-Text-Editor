# Yardım Sistemi (Help System)

Kullanıcıların takıldığı noktalarda başvurabileceği bilgi kaynağı.

## Özellikler

*   **Kısayol Listesi:** Tüm klavye kısayollarını listeler.
*   **Dokümantasyon:** Özellikler hakkında detaylı açıklamalar sunar.
*   **Hakkında:** Uygulama sürümü ve geliştirici bilgilerini içerir.

## Kullanım

F1 tuşu veya Yardım menüsü üzerinden erişilebilir.
