# Sözdizimi Vurgulayıcı (Syntax Highlighter)

Kodun okunabilirliğini artırmak için farklı kod bileşenlerini renklendiren modül.

## Özellikler

*   **Renk Temaları:** Seçilen temaya uygun renklendirme.
*   **Dil Algılama:** Dosya uzantısına göre otomatik dil tespiti.
*   **Kapsamlı Destek:** Python, JavaScript, HTML, CSS ve daha birçok dil desteği.

## Kullanım

Dosya açıldığında veya dil değiştirildiğinde otomatik olarak devreye girer.
